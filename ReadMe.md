递归版本： 关键点在于下面两句的理解，看来递归的样式很多，写出来的代码很简单，过程却有点难以理解
head.next.next = head
head.next = None

``
def reverseList(self, head: ListNode) -> ListNode:
        if head is None or head.next is None:
            return head
            
        result = self.reverseList(head= head.next)
        head.next.next = head
        head.next = None
    
        return result
``
